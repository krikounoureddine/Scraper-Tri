Description des Technologies

🎨 Couche Présentation

  • Tkinter: Interface utilisateur desktop native
  • Folium: Cartes interactives et visualisations géospatiales
  • python-docx: Génération automatique de documents Word
  • openpyxl: Création et manipulation de fichiers Excel

⚙️ Couche Logique Métier

  • Python: Langage de programmation principal
  • Selenium: Automatisation des navigateurs web
  • BeautifulSoup: Analyse et parsing HTML/XML
  • Requests: Requêtes HTTP et APIs REST

💾 Couche Données

  • SQLite: Base de données relationnelle légère
  • XML: Échange et stockage structuré de données
  • JSON: Configuration et données légères
  • CSV: Export/import de données tabulaires

🛠️ Couche Infrastructure

  • Git & GitHub: Gestion de versions et collaboration
  • VS Code: Environnement de développement
  • Trello: Gestion de projet agile
